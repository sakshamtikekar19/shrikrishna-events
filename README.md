# Shree Krishna Event Management

Luxury event management website for Shree Krishna Event Management — Bhubaneswar.

**Live site:** https://sakshamtikekar19.github.io/shrikrishna-events/

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy (GitHub Pages)

This project builds as a static export. On every push to `main`, GitHub Actions builds and deploys to GitHub Pages.

In the repo: **Settings → Pages → Build and deployment → Source: GitHub Actions**.

## Stack

Next.js (App Router), React, TypeScript, Tailwind CSS, Framer Motion
